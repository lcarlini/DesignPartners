The catalog

Fourteen partners, three families

Grouped the classic way — creational, structural, behavioral — each with its own design pressure.

Creational

How objects come to life

Control construction without leaking complexity.

  • Abstract FactorySwap regional payment/tax families as one unit
  • BuilderAssemble complex orders without telescoping constructors
  • Factory MethodDefer channel creation while sharing notification flow
  • PrototypeClone product blueprints into cheap variants
  • SingletonShare configuration safely across collaborators
Structural

How objects compose

Wire pieces together and hide the seams.

  • AdapterMake a legacy freight API speak a modern contract
  • FacadeCollapse checkout orchestration behind one entry point
  • ProxyCache expensive catalog lookups transparently
  • CompositeTreat items and nested bundles the same way
Behavioral

How objects collaborate

Coordinate responsibility and change over time.

  • StrategySwap discount algorithms without conditional sprawl
  • ObserverFan out order events to independent subscribers
  • CommandCapture undoable inventory mutations
  • StateOwn valid order lifecycle transitions
  • Template MethodReuse report structure with customizable steps
Project as example

One pattern, from the real repo

Here is Abstract Factory in DesignPartners — regional checkout without a maze of conditionals.

Swap a whole family, not a branch

CheckoutService depends only on ICommerceFactory. Choose DomesticCommerceFactory or the international one, and both the payment gateway and tax calculator switch together — the checkout code never changes.

ICommerceFactory Money CheckoutResult
src/DesignPartners/Creational/AbstractFactory.cs 
public sealed class CheckoutService(ICommerceFactory factory)
{
    public CheckoutResult Checkout(string customerId, Money subtotal)
    {
        // payment + tax always come from the same family
        var tax = factory.CreateTaxCalculator().Calculate(subtotal);
        var total = new Money(subtotal.Amount + tax.Amount, tax.Currency);
        var receipt = factory.CreatePaymentGateway().Charge(total, customerId);
        return new CheckoutResult(subtotal, tax, total, receipt);
    }
}
